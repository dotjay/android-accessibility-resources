## TalkBack commands

Use the core keyboard commands above plus the following shortcuts to access TalkBack features.

Action              |  Shortcut
--------------------|-----------------
Global context menu | <kbd>alt</kbd> + <kbd>space</kbd>
Local context menu  | <kbd>alt</kbd> + <kbd>shift</kbd> + <kbd>space</kbd>
